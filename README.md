# IT Jobs Statistics

Веб-сервис анализа рынка IT-вакансий. Первый этап реализован UI-first: Flask + Jinja2 + Chart.js с реалистичными демо-агрегатами.

## Что уже есть

- Главная страница с фильтром по специальности.
- Метрики: вакансии, активные резюме, коэффициент конкурентности.
- Графики: грейды, требуемый опыт, формат работы.
- Таблица топ-навыков.
- Fallback для направлений без вакансий.

## Локальный запуск

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python -m app.main
```

Откройте `http://127.0.0.1:5000`.

## Тесты

```bash
pytest
```
